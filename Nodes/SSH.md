# Проброс портов
На другое устройство
```bash unfold
ssh -L [локальный_порт]:[целевое_устройство]:[порт_целевого_устройства] [логин_сервера]@[адрес_сервера]
ssh -L 8080:192.168.1.1:80 daniil@10.30.111.115

 # In browser http://localhost:8080/
```
Проброс на python сервер для просмотра локальных файлов
```bash unfold
python3 -m http.server 8000
ssh -L 8080:localhost:8000 user@remote-server
```
