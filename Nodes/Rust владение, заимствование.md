* scope - область видимости
* borrowing - заимствование
* ownership - владелец

Простые типы копируются. ***Сложные типы (у которых длина заранее неизвестна) move'ятся.***
```rust unfold
let a = 42; // а владеет числом 42. Когда a будет уничтожено, данные будут уничтожены
```
### Ownership
```rust unfold
let a = 42;
let b = a; // Значение "a" копируется

let s1 = String::from("Hello"); // s владеет данными
let s2 = s1; // Владение "Hello" передано s2. Обратиться к s1 больше нельзя.
let s3 = s2.clone(); // копирование

fn func(s: String) -> String {
	// ...
	s
} // Если владение не вернуть из функции, по выходу из функции значение будет уничтожено

fn func2(mut s: String) -> String {
	// ...
	s
} 

let s3 = func(s3); // Передали неизменяемое владение в функцию и вернули обратно
let s3 = func(s3); // Передали изменяемое владение
```
### Borrowing
***В любой момент может быть либо любое количество неизменяемых заимствований, либо изменяемое, но толко одно***
```rust unfold
let mut s = String::from("Hello");

fn func(s: &String) {
	// ...
}

fn func2(s: &mut String) {
	// ...
}

func(&s); // Передали неизменяемое заимствование. s3 все еще владеет переменной
func(&mut s); // // Изменяемое заимствование
```

### Slice (нельзя изменять даже с `mut`)
* `&str` литерал (отличается, потому что у такого неограниченное время жизни)
```rust unfold
let s: &str = "test";
```
* `&str` - срез строки
```rust unfold
let s = String::from("Hello");
let sl: &str = &s1[0..3];
```
* `&[i32]` - срез массива (и так далее по типам)
```rust unfold
let a = [1, 2, 3, 4, 5];
let nice_slice = &a[1..=3; // 2,3,4
let nice_slice = &a[1..3]; // 2,3
```
